My website portfolio is divided into two sections, a navigation bar, and contact footer

- Contains my logo/name, and quick hyperlinks to corresponding sections of my website profile

ABOUT ME SECTION
- Contains my personal profile picture
- Contains an about me section
- The right half of the page contains my work experiences

PROJECTS SECTION
- Contains projects worked on previously
- Also showcases details and features of projects worked on

CONTACT FOOTER
- Contains personal contact details (email, phone number, and github)
